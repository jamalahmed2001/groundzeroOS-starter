# Agent Memory Hub

> Canonical archive of all OpenClaw agent memory. Daily notes are staged in the workspace `memory/` directory and synced here by `dailySync`. Each month is consolidated into a single overview of key decisions, principles, and wisdom.

## How Memory Flows

1. OpenClaw writes session notes to `~/clawd/memory/YYYY-MM-DD.md` (workspace staging)
2. `dailySync` appends daily content into the current month's consolidated file here
3. `MEMORY.md` is the curated long-term summary — updated periodically
