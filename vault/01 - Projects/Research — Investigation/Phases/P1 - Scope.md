# P1 — Scope

## Overview

Pin the question, decompose it into sub-questions you'll actually answer, draft an initial source list, name what's out of scope. The deliverable is a one-page scope doc, not source-fetching.

## Tasks

- [ ] Restate the research question in your own words. If you can't restate it cleanly, the question isn't ready.
- [ ] Decompose into 3-5 sub-questions, each answerable from sources.
- [ ] Estimate the source count needed per sub-question (peer-reviewed, trade press, primary, etc. per the project's `source_constraints`).
- [ ] Seed an initial source list — 5-10 candidates per sub-question, by name (search later).
- [ ] List what's *out of scope* — adjacent questions you won't answer in this investigation.
- [ ] Define what "answered" looks like — a confidence threshold per sub-question.
- [ ] Write `Phases/P1 - Scope - Decisions.md` with all of the above.

## Acceptance Criteria

- [ ] `Phases/P1 - Scope - Decisions.md` exists.
- [ ] Question + 3-5 sub-questions stated.
- [ ] Source count target named per sub-question.
- [ ] Out-of-scope explicit.
- [ ] No actual source fetching done.
