## Acceptance gates

The `research` profile's gate is "source count + gaps addressed". Each phase tightens it:

| Phase | Acceptance |
|---|---|
| P1 Scope | Decomposed question; source list seeded; out-of-scope explicit. |
| P2 Gather | ≥ N sources fetched (declared in P1); each tagged by quality. |
| P3 Synthesise | One-page synthesis; agreement / disagreement among sources mapped; gaps named. |
| P4 Write | Deliverable written in declared `output_format`; every claim cited inline; gaps explicit. |
| P5 Review | External (or independent-of-author) sign-off; revisions integrated. |
