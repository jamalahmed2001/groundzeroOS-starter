# A01 O1 — Concept + Tracklist

## Overview

Pin the album's single surface metaphor + timeless human truth underneath, then derive the tracklist from it. Track titles concrete; the truth never named in lyrics. Each track varies within the core palette per the Style Guide.

## Tasks

- [ ] Update the album's `## Tracklist` table.
