# A01 O5 — Metadata + SEO

## Overview

Per-track and per-album metadata. Genre / mood per the distributor's vocabulary. ISRCs (if pre-assigned). Songwriter / featured-artist credits. Explicit flags. Release date.

## Tasks

- [ ] Apply [[08 - System/Agent Directives/metadata-curator.md|metadata-curator]].
- [ ] For each track: title, primary genre, mood / sub-genre, songwriter credit, featured artists (if any), ISRC (if pre-assigned), explicit flag, release date.
- [ ] For the album: title, primary genre, secondary genres, mood, language, copyright line, P-line, UPC (if pre-assigned).
- [ ] Honour the distributor's genre vocabulary — don't invent genres the platform doesn't have.
- [ ] Update the album's `## Metadata` section with per-track + album-level metadata, and a per-platform variant (Spotify / Apple / etc. each have slightly different field names + length limits).

## Acceptance Criteria

- [ ] Metadata complete for every track + the album.
- [ ] All genres / moods honour the distributor's vocabulary.
- [ ] No fabricated credits or ISRCs.
