# A01 O7 — Playlist Pitching

## Overview

Pitch the lead track to streaming-platform editorial + relevant independent curators. The window is ≥ 1 week before release for editorial; rolling for independents.

## Tasks

- [ ] Apply [[08 - System/Agent Directives/launch-ops.md|launch-ops]].
- [ ] Pitch the lead track via Spotify for Artists' "Pitch a song" flow (or distributor-native pitching).
- [ ] Compile a list of relevant independent playlists (genre-matching, audience-matching).
- [ ] Draft pitch emails / DMs to each independent curator. Personalise — generic pitches get ignored.
- [ ] Send (or queue for operator approval if scope allows it).
- [ ] Track responses in `Albums/<album>/Pitches.md`.

## Acceptance Criteria

- [ ] Editorial pitch submitted (or noted as "submitted via distributor" if distributor handles it).
- [ ] ≥ N independent pitches sent (N depends on artist's outreach scope).
- [ ] All sent pitches logged in Pitches.md.
